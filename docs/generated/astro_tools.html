  • obj – array with positions in SkyCoord coordinated

astro_tools.is_within_angle(obj_alt, obj_az, point_alt, point_az, within_angle)[source]¶

Determine if a position is hidden by some object

Parameters

  • obj_alt – interfering object’s position (alt)

  • obj_az – interfering object’s position (az)

  • point_alt – position of point being observed (alt)

  • point_az – position of point being observed (az)

  • within_angle – defines zone around center of interfering object to avoid

Returns

True if there is interference, False otherwise

astro_tools.is_hidden_by_moon(moon, point, location, obstime, within_angle=0.25)[source]¶

Determine if a position is hidden by the moon

Parameters

  • moon – approximate coordinates of the moon, as returned by astropy.coordinates.get_body

  • point – sky position being observed (in SkyCoord coordinates)

  • location – EarthLocation from which observation is made (eg, Kiso)

  • obstime – observation time

  • within_angle – defines zone around center of interfering object to avoid

Returns

True if there is interference, False otherwise
