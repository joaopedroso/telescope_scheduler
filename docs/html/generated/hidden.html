hidden¶

hidden: Indicate positions unavailable for observations

Function hidden(sky) is called before each observation to indicate positions not currently available, typically due to being concealed by clouds

Functions

hidden(sky)

Return indices in sky list that are currently hidden by clouds or other objects
hidden.hidden(sky)[source]¶

Return indices in sky list that are currently hidden by clouds or other objects

Note

sky list positions hidden by the moon are removed during preprocessing

Parameters

sky – list of SkyCoord positions to be observed

Returns

indices of sky that cannot be observed now

Return type

sequence of int
